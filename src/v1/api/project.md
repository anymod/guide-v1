You can get the current project at any time with `Anymod.Project`. For example, a `<script>` tag with:

```html
<script project="8N3P38" src="https://cdn.anymod.com/v1"></script>
```

Would yield the following:

```js
Anymod.Project
// -> "8N3P38"
```
