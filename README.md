## Anymod Guide
This app runs the guide based on Hexo, with app located at:

| NODE_ENV | URL |
|---|---|
| `production`  | https://guide.anymod.com |
| `development`  | http://localhost:*TBD* |

## Operations

### Run local server
`hexo server`

### Clean local files
`hexo clean`

### Regenerate local files
`hexo generate`

### Deploy to live S3
`hexo generate`
